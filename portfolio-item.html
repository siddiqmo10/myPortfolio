SlackClone

My clone of the slack application built with react

This is a clone of the popular business communication platform that I made using ReactJS to handle the functionality. I've also added google user authenthication using firebase's easy to set up google authentication which allows users to sign in with their google accounts.

I've used React's Context API in order to fetch data from the database. I've also styled the web app using Material UI accompanied with CSS Varibales. And to add the final touches I used Flex Box for giving the web app the well organized professional look.

One problem that I'm currently working on, is the functionality to switch over to different channels which I think I can solve with the use of react-router. I'm using a real time database by firebase which is also how I'm deploying my web app.
